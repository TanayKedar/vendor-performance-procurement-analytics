# Vendor Performance Analysis

End-to-end data analytics project using SQL, Python, and Power BI.

## Project Overview
Analyzed vendor purchase and sales data to identify profitability, top performers, low-performing vendors, inventory inefficiencies, and business opportunities.

## Tech Stack
- MySQL
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Power BI
- DAX

## Key Features
- Data ingestion from CSV to MySQL
- Data cleaning and feature engineering
- Exploratory Data Analysis
- Statistical testing (T-Test)
- Power BI interactive dashboard
- Vendor & Brand performance insights

## KPIs
- Total Sales
- Total Purchase
- Gross Profit
- Profit Margin
- Unsold Capital

## Dashboard Includes
- Purchase Contribution Donut
- Top Vendors & Brands
- Low Performing Vendors
- Target Brands Identification
- Inventory Analysis
